\section*{Introduction: Why Change Variables?}

The primary utility of the Fundamental Theorem of Calculus is the ease with which we can compute definite integrals. Even with this powerful tool, some integrals are far more difficult (or impossible) than others.

Changing variables is one tool we have for making difficult integrals much easier to compute.

In single-variable calculus we called this ``u-substitution''. We now more generally use various strategic coordinate systems to simplify our calculations by taking advantage of nice geometries.

\begin{problem}
Recall u-substitution from Calculus I:
$$\int f(g(x)) g'(x)\,dx = \int f(u)\,du \text{ where } u = g(x)$$

Example: $$\int_0^1 \frac{2x}{(1+x^2)^2}\,dx$$

A much easier integral to evaluate is $$\int_1^2 \frac{1}{u^2}\,du,$$ which we can get if we change variables from $x$ to $u$ using the coordinate change $u = 1 + x^2$.

What is difficult about this integral that keeps us from being able to easily apply the Fundamental Theorem of Calculus?
\begin{multipleChoice}
    \choice{The function is not continuous}
    \choice{The function is not integrable}
    \choice[correct]{$\frac{2x}{(1+x^2)^2}$ does not have a nice antiderivative}
    \choice{It's not a difficult integral}
\end{multipleChoice}

Indeed, we can either: 1) memorize every possible antiderivative, or 2) we can gain tools to transform difficult integrals into easier ones.

Option 2 is much more flexible.

Let's review why variable changes work the way they did in single-variable calculus.

If $u=g(x)$, what is wrong with the substitution $$\int f(g(x))\,dx = \int f(u)\,du$$ (that is, why do we need $g'(x)$ in the substitution)?

\begin{multipleChoice}
    \choice{$g'(x)$ tells us the new bounds of integration}
    \choice{Without $g'(x)$, we would be integrating with respect to the wrong variable}
    \choice[correct]{Because of the chain rule, $d(f(u))=f'(u)du = f'(g(x))g'(x)dx$, so we need $g'(x)$ to account for how $u$ changes with $x$}
    \choice{Nothing is wrong with that substitution}
\end{multipleChoice}


\end{problem}

If we change variables, we also need to know how to re-compute the differential elements in the new variable system. The same is true in multiple integrals: 

When we change coordinate systems, we need to know how to compute the new area and volume elements $dA$ and $dV$.

\textbf{Review: The Cartesian Approach}

Last module, we set this up in rectangular coordinates with order $dz\,dy\,dx$.

The region $x^2 + y^2 \leq 1$ and $0 \leq z \leq 4-x^2-y^2$ left us with the following bounds of integration:

\begin{itemize}
    \item $z$ bounds: $0 \leq z \leq 4-x^2-y^2$
    \item $y$ bounds: $-\sqrt{1-x^2} \leq y \leq \sqrt{1-x^2}$

\end{itemize}

This gave us the following triple integral:

$$M = \int_{-1}^{1} \int_{-\sqrt{1-x^2}}^{\sqrt{1-x^2}} \int_0^{4-x^2-y^2} z\,dz\,dy\,dx$$

Which became messy after we integrated with respect to $z$
$$M = \int_{-1}^{1} \int_{-\sqrt{1-x^2}}^{\sqrt{1-x^2}} \frac{(4-x^2-y^2)^2}{2}\,dy\,dx$$

\subsection*{The Polar Area Element}

Much like changing between the differentials $dx$ and $du$ required attention to the chain rule, in higher dimensions we need to make sure that the differential area $dA$ accounts for the geometry of the coordinates \emph{and} allows us to use iterated integrals.

\begin{problem}
What is $dA$ in polar coordinates?


We're going to use the term \emph{parameter space} to refer to the simple rectangular grids that make up the variables we're integrating in.

For iterated integrals, we need to use rectangular geometries. So we need the final differential to be some multiple of $dr\,d\theta$. The problem is that $dr\,d\theta$ does not compute the area of a small polar region because the coordinate geometry is different.

So we need to figure out how to compute the area of a small polar region in terms of $dr$ and $d\theta$.

In polar regions, the curved arc lengths depend on the size of the radius. So one arc length is $ r\,\Delta\theta$. 

If $r_1$ is the inner radius of a polar area and $r_2$ is the outer radius, then we can approximate the area of the region using the average $\Delta A\approx \frac{r_1 + r_2}{2} \cdot \Delta r \cdot \Delta\theta$. 

As we make the region small enough for a linear approximation, $r_1$ and $r_2$ get closer together and we can use the differential

$$dA = r\,dr\,d\theta.$$


The extra factor $r$ is important because:

\begin{multipleChoice}
    \choice{The integral is easier}
    \choice[correct]{The geometry of the coordinates is different, so we need to account for how the area elements change}
    \choice{Nothing would change}
\end{multipleChoice}

When we integrate in polar coordinates, the function being integrated becomes a ``height'' and so the product $f\cdot dA$ looks like a ``cylindrical wedge''. We compute $dV$ by multiplying the differential area $dA$ by the height $f$, giving $dV=f\cdot dA = f\cdot r\,dr\,d\theta$.

\begin{problem}
\textbf{Finishing the Mass Integral of the curved-top cylinder}

Now that we know how to translate $dA$ between $dx\,dy$ and $dr\,d\theta$, we can finish the mass integral from the previous problem.

Most of the complexity before came from $x$ and $y$, after we integrated in $z$.

So we make the integral simpler by changing the geometry of the $xy$ plane from Cartesian to polar cooridnates, leaving $z$ alone.

In the $xy$-plane the area element $dx\,dy$ becomes $\answer{r}\,dr\,d\theta$.  Since $z$ does not change, the full 3D volume element is:

$$dV = dx\,dy\,dz = \answer{r}\,dr\,d\theta\,dz$$

\textbf{Step 1: Transform the Region}

The base $x^2 + y^2 \leq 1$ becomes: $\answer{r^2} \leq 1$, so $\answer{0} \leq r \leq \answer{1}$

The solid is a full rotation around, meaning $\answer{0} \leq \theta \leq \answer{2\pi}$

The paraboloid $z = 4 - x^2 - y^2$ becomes: $z = \answer{4 - r^2}$

So: $0 \leq z \leq \answer{4-r^2}$

\textbf{Step 2: Transform the Density}

The density was $\rho(r,\theta,z) = \answer{z}$

\textbf{Step 3: Set Up the Integral in Cylindrical Coordinates}

$$M = \int_{\answer{0}}^{\answer{2\pi}} \int_{\answer{0}}^{\answer{1}} \int_{\answer{0}}^{\answer{4-r^2}} \answer{z\cdot r}\,dz\,dr\,d\theta$$

\textbf{Step 4: Evaluate the Integral}

\textbf{Inner integral ($z$):}
$$\int_0^{4-r^2} zr\,dz = r\left[\frac{z^2}{2}\right]_0^{4-r^2} = r\cdot \answer{\frac{(4-r^2)^2}{2}}$$


\textbf{Middle integral ($r$):}

Whereas this integral still requires a little bit of algebra, it is far more easy to calculate than the integral we were working with last module.

Go ahead and go through the algebra (either some distributing or a simple u-substitution) to evaluate this integral:

$$\int_0^1 \frac{r(4-r^2)^2}{2}\,dr=\answer{[tolerance=.01]\frac{37}{12}}$$

\textbf{Outer integral ($\theta$):}
$$M = \int_0^{2\pi} \frac{37}{12}\,d\theta = \answer[tolerance=.01]{\frac{37\pi}{6}}$$

Notice that the final integral was actually very easy because the bounds independently relied on variation in $\theta$ and $r$, whereas in cartesian the variation always intermingled $x$ and $y$.

This is one example where polar cooridnates make the use of the FTC very easy.
\end{problem}

\begin{problem}
Let's integrate $f(x,y) = xy$ over the top half of the unit disk $x^2 + y^2 \leq 1$, $y \geq 0$.

\textbf{In rectangular coordinates:}
$$\int_{-1}^1\int_{-\sqrt{1-x^2}}^{0} xy\,dy\,dx$$

\textbf{In polar coordinates:}
\begin{itemize}
    \item Region: $\answer{0} \leq r \leq \answer{1}$, $\answer{0} \leq \theta \leq \answer{\pi}$
    \item Remember the transformation: $x = r\cos\theta$, $y = r\sin\theta$
    
    Function: $f(x,y) = xy = (r\cos\theta)(r\sin\theta) = \answer{r^2 \sin\theta \cos\theta}$
    \item Area element: $dA = \answer{r}\,dr\,d\theta$
\end{itemize}

The integral becomes:
$$V = \int_\answer{0}^{\answer{\pi}}\int_0^1 \answer{r^3\sin\theta\cos\theta}\,dr\,d\theta$$

\textbf{Inner integral (with respect to $r$):}
$$\int_0^1 r^3\sin(\theta)\cos(\theta)\,dr = \sin(\theta)\cos(\theta)\left[\frac{r^4}{4}\right]_0^1 = \answer{\frac{1}{4}\sin\theta\cos\theta}$$

\textbf{Outer integral (with respect to $\theta$):}
You can do the outer integral a couple of ways, with a simple u-substitution or a trig integral, or a double angle formula. Either way, this is a fairly straightforward integral from your past:

$$\int_0^{\pi} \frac{1}{4}\sin\theta\cos\theta\,d\theta  = \answer{0}$$

Why is the volume zero?
\begin{multipleChoice}
    \choice{We made a calculation error}
    \choice[correct]{The negative and positive volumes cancel out due to the symmetry of $xy$}
    \choice{The unit disk has zero area}
\end{multipleChoice}
\end{problem}
